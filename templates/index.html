Sign Recognition

AI Sign Recognition is a web application that uses machine learning to recognize hand gestures, Indian Sign Language (ISL) and traffic signs. It uses Convolutional Neural Networks (CNN) to classify the images. The application is built using Flask, a Python web framework.
